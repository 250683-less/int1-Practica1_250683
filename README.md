## Comandos basicos para maquetado de la documentacion utilizando el estandar de markdown (.md)
es un lenguaje de marcado ligero que te permite dar formato a textos usando símbolos simples en un editor de texto plano (como el Bloc de notas, o editores dedicados). Se convierte automáticamente a HTML y es ampliamente utilizado en plataformas como GitHub, Discord, Notion y foros web

### 1. Encabezado o titulos (HEADERS)
Para realizar una buena documentacion del proyecto debemos distribuir correctamente los contenidos, para poder delimitar o hacer enfasis (enfatizar), es decir resaltar las seciones mas importantes, podremos utilizar lo siguiente

**Ejemplos**

#Encabezado nivel 1
## Encabezado nivel 2
### Encabezado nivel 3
#### Encabezado nivel 4
##### Encabezado nivel 5
###### Encabezado nivel 6
####### Encabezado nivel 7 - *El estandar solo permite 6 niveles para titulos, a partir del septimo seran presentados como texto plano (sin estilo)*

### 2. Separadores (SEPARATIORS)
si se desea marcar una separacion visual de los contenidos podemos utilizar unq linea horizontal indicando tres caracteres
